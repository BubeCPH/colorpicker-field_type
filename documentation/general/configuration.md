# Configuration

**Example Definition:**

```
protected $fields = [
    'example' => [
        'type'   => 'anomaly.field_type.colorpicker',
        'config' => [
            'default_value' => null
        ]
    ]
];
```

### `default_value`

The default color for the colorpicker. Any valid hexadecimal color value can be used. No default value is specified by default.
